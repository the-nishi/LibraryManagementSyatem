### 📚 Library Management System (Java)

A Java-based application designed to manage library operations such as book records, user management, and borrowing/returning processes. This project applies Object-Oriented Programming concepts to simulate a real-world library system.

#### 🚀 Features

* Add, update, and manage book records
* User/member management system
* Borrow and return book functionality
* Organized data handling using classes
* Modular and maintainable code structure

#### 🛠️ Tech Stack

* **Language:** Java
* **Paradigm:** Object-Oriented Programming (OOP)

#### 📂 Project Structure

* `Book/` – Handles book-related operations
* `User/Member/` – Manages user data
* `Transaction/` – Borrowing and returning logic
* `Main.java` – Application entry point

#### 🎯 Purpose

This project was developed to:

* Simulate a real-world management system
* Apply OOP concepts in a practical scenario
* Improve system design and problem-solving skills
